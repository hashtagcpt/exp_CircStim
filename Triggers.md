Triggers
========

stimuli
-------

fixation   - `1`  
mask       - `2`  
target (`4`-`7`):  
  - 0   deg - `4`
  - 45  deg - `5`
  - 90  deg - `6`
  - 135 deg - `7`

there is no marker for behavioral response, reaction times are saved so marker would be redundant  

blocks
------
training start  - `10`  
fitting  start  - `20`  
contrast part   - `30`  
time part start - `40`  

breaks
------
start - `100`  
stop  - `102`
